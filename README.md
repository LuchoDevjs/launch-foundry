# LaunchFoundry

A reusable web factory for premium landing pages and ecommerce storefronts.

## What this is

LaunchFoundry is the base repo for building polished client websites quickly:

- premium landing pages,
- ecommerce storefronts connected to Tiendanube,
- reusable sections and templates,
- deploy-ready Next.js app.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Motion for React
- Vercel

## Local commands

```bash
npm install
npm run dev
npm run build
npm run lint
npm run typecheck
```

## Deployment

Create a GitHub repo, push this project, then import it into Vercel.
